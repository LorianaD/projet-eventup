## 🧭 Overview

**EventUP** est un projet personnel développé dans une logique d’apprentissage avancé du développement web fullstack.

Il a été conçu pour explorer :

* la structuration d’une application complète
* la gestion des utilisateurs
* l’upload et la gestion de médias
* la factorisation et la maintenabilité du code

---

## 🎯 Learning Goals

Ce projet m’a permis de consolider et expérimenter :

* 🔐 Systèmes d’authentification (login / register)
* 📁 Upload de fichiers (images & vidéos)
* 🧱 Architecture fullstack claire et évolutive
* 🔄 Communication API (frontend ↔ backend)
* 🎨 Design moderne avec Tailwind & DaisyUI
* 📦 Organisation des assets et des composants

## 🚀 Getting Started

### 1. Clone repository

```bash
git clone https://github.com/LorianaD/projet-eventup.git
cd projet-eventup
```

---

### 2. Frontend

```bash
cd front
npm install
npm run dev
```

---

### 3. Backend

```bash
cd back
npm install
npm run dev
```

---

### 4. Environment variables

Créer un fichier `.env` dans le backend :

```env
PORT=5000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_NAME=eventup
JWT_SECRET=your_secret_key
```
